# Hotel Booking API

🏨 Backend для системы бронирования отелей

## Tech Stack

- Node.js + Express.js + TypeScript
- MySQL + Sequelize ORM
- JWT + bcryptjs
- Joi validation

## Installation

```bash
npm install
```

## Running

```bash
npm run dev     # Development
npm run build   # Build
npm start       # Production
```

## API Endpoints

- `GET /health` - Проверить что сервер живой
- `GET /` - Главная страница
